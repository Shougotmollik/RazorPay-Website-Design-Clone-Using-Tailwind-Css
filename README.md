## Deployment

To deploy this project run


- prerequisite 
    ```bash
  Your System Must Have Node jS Installed.
    ```
 1 . First Clone project


```bash
  https://github.com/Shougotmollik/RazorPay-Website-Design-Clone-Using-Tailwind-Css.git
```

2 . Run The Project 
```bash
  npm run start
```
